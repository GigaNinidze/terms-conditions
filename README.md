# Terms and Conditions Website

A simple 2-page website for hosting Terms and Conditions and Data Deletion information for a WhatsApp bot on GitHub Pages.

## Pages

- **index.html** - Terms and Conditions page
- **deletion.html** - User Data Deletion instructions

## Setup for GitHub Pages

1. **Create a GitHub repository**
   - Go to GitHub and create a new repository
   - Name it `termsandconditions` (or any name you prefer)

2. **Upload files**
   - Upload all files (index.html, deletion.html, styles.css) to the root of your repository

3. **Enable GitHub Pages**
   - Go to your repository Settings
   - Navigate to Pages section
   - Under "Source", select the branch (usually `main` or `master`)
   - Select the `/ (root)` folder
   - Click Save

4. **Access your site**
   - Your site will be available at: `https://[your-username].github.io/termsandconditions/`
   - It may take a few minutes for the site to be live

## Customization

- **Email address**: Update the email `giga.ninidze@icloud.com` in both HTML files if needed
- **Content**: Modify the terms and conditions text in `index.html` as needed
- **Styling**: Adjust colors and spacing in `styles.css` using the CSS variables at the top
